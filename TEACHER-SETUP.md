# Teacher Setup Guide — GitHub Design Portfolio

## How to set up for your class

### Option A: GitHub Classroom (Recommended)
1. Create a GitHub organisation for your school (e.g. `aphs-multimedia`)
2. Push this `design-portfolio-template` folder as a **template repository**
3. Set up a **GitHub Classroom** assignment:
   - Go to https://classroom.github.com
   - Create a new assignment
   - Choose this template repo
   - Share the assignment link with students
4. Each student gets their own private copy automatically
5. Students edit `README.md` directly in the browser (click the pencil icon)
6. Every save = a git commit (auto-saved history)
7. Students submit by sharing their repo URL on Google Classroom

### Option B: Manual Fork Method
1. Push this folder to a public GitHub repo
2. Students fork the repo to their own accounts
3. They edit `README.md` in the browser
4. Submit by sharing their fork URL

### How students add images
1. In their repo, click the `images/` folder
2. Click "Add file" > "Upload files"
3. Drag and drop screenshots
4. In `README.md`, the image paths like `images/wireframe-1.png` will automatically display

### Grading
- View each student's repo to see their portfolio
- Check commit history to see their progress over time
- The README renders as a formatted page on GitHub — easy to read and grade
